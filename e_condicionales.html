Problemas Condicionales en Python

Ejercicios para Bachillerato de Sociales

Enunciados

1. Cálculo de Edad

Escribe un programa que le pida al usuario que introduzca su año de nacimiento y calcule su edad actual. Asegúrate de que el programa solo permita el acceso a personas mayores de 18 años. Si el usuario tiene menos de 18, el programa debe mostrar un mensaje indicando que no puede continuar.

2. Adivina el Número

Escribe un programa en el que el ordenador elija un número al azar entre 1 y 10, y le pida al usuario que intente adivinarlo. Si el usuario adivina correctamente, el programa debe mostrar "¡Correcto!". Si no, debe mostrar "Sigue intentando". Pista: Usa una variable para almacenar el número secreto, y utiliza condicionales para comparar el número ingresado con el número secreto.

3. Acceso por Contraseña

Escribe un programa que le pida al usuario ingresar un nombre de usuario y una contraseña. Si el nombre de usuario es "admin" y la contraseña es "1234", el programa debe mostrar "Acceso concedido". Si no, debe mostrar "Acceso denegado". Este tipo de programa se utiliza en muchos sitios web y sistemas para controlar el acceso.

4. Mayor de Tres Números

Escribe un programa que le pida al usuario que ingrese tres números. El programa debe determinar cuál de los tres números es el mayor. Usa condicionales para comparar los tres valores y asegúrate de considerar los casos en los que dos o más números pueden ser iguales.

5. Conversión de Metros a Diferentes Unidades

Escribe un programa que le pida al usuario una distancia en metros. Luego, el programa debe preguntarle en qué unidad quiere convertir esa distancia: kilómetros, centímetros o milímetros. El programa debe mostrar el resultado de la conversión. Usa la siguiente tabla para las conversiones:

  • Kilómetros = metros ÷ 1000
  • Centímetros = metros × 100
  • Milímetros = metros × 1000
El programa debe mostrar un mensaje de error si el usuario ingresa una opción no válida.

6. Número Par o Impar

Escribe un programa que le pida al usuario que introduzca un número y determine si es par o impar. Un número es par si al dividirlo entre 2, el resto es 0. Usa el operador % (módulo) para verificar si el número es divisible entre 2 sin dejar resto.

7. Calificación de un Examen

Escribe un programa que le pida al usuario una calificación de examen entre 0 y 100. Según la calificación ingresada, el programa debe mostrar una evaluación:

  • 90 o más: Sobresaliente
  • 70-89: Notable
  • 50-69: Aprobado
  • Menos de 50: Suspenso
Usa varias condiciones para mostrar el mensaje adecuado.

8. Año Bisiesto

Escribe un programa que pida al usuario un año y determine si es un año bisiesto. Un año es bisiesto si cumple una de las siguientes condiciones:

  • Es divisible por 4, pero no por 100.
  • O, es divisible por 400.
Usa condicionales para verificar ambas condiciones.
