TICO.2.A.1. Programación en Python

Estructura de un programa en Python:

Un programa en Python típicamente sigue una estructura secuencial donde las instrucciones se ejecutan de arriba hacia abajo. Python es un lenguaje interpretado, lo que significa que cada línea de código se ejecuta en el orden en que aparece.

Elementos básicos del lenguaje incluyen:

  • Comentarios: Se usan para documentar el código. Comienza con # para una sola línea o ''' ''' para bloques múltiples.
  • Variables: Almacenan valores. No se declaran tipos explícitamente. Ejemplo: edad = 25
  • Expresiones: Combinación de operadores y operandos. Ejemplo: 3 + 4 * 2
Tipos de Lenguajes

Existen distintos tipos de lenguajes de programación:

  • Lenguajes de alto nivel: Python es un ejemplo. Fácil de leer, escribir y mantener.
  • Lenguajes de bajo nivel: Como el ensamblador, están más cerca del hardware.

Tipos de datos:
  • int: Enteros. Ejemplo: num = 5
  • float: Números con decimales. Ejemplo: pi = 3.14
  • str: Cadenas de texto. Ejemplo: nombre = "Juan"
  • bool: Valores booleanos. Ejemplo: es_valido = True
Constantes y Variables:

En Python, las variables no necesitan declaración explícita de tipo. Las constantes no son estrictamente impuestas, pero se nombran en mayúsculas por convención. Ejemplo: PI = 3.1416.

Operadores:

Algunos ejemplos de operadores:

  • Aritméticos: +, -, *, /, // (división entera), % (módulo)
  • Relacionales: ==, !=, >, <, >=, <=
  • Lógicos: and, or, not
Entrada/Salida:

La función input() se usa para la entrada de datos. Ejemplo: nombre = input("Introduce tu nombre: ")

Para la salida de datos se utiliza print(). Ejemplo: print("Hola, " + nombre)

Estructuras Condicionales:

Las estructuras condicionales permiten tomar decisiones en el programa. La estructura básica es:

if condicion:
# Código si la condición es verdadera
elif otra_condicion:
# Código si la otra condición es verdadera
else:
# Código si ninguna condición es verdadera
Estructuras Iterativas:

Se usan para ejecutar un bloque de código repetidamente. Ejemplos:

# Bucle while
while condicion:
# Código a ejecutar mientras la condición sea verdadera

# Bucle for
for i in range(10):
print(i)

range() genera una secuencia de números, útil para contar iteraciones.

Principales estructuras de datos en Python:
  • Listas: Permiten almacenar múltiples elementos. Ejemplo: mi_lista = [1, 2, 3, 4]
  • Tuplas: Son similares a las listas, pero inmutables. Ejemplo: mi_tupla = (1, 2, 3)
  • Diccionarios: Almacenan pares clave-valor. Ejemplo: mi_diccionario = {"nombre": "Juan", "edad": 25}
  • Conjuntos (Sets): Almacenan valores únicos y no ordenados. Ejemplo: mi_set = {1, 2, 3}

Las estructuras de datos son fundamentales para organizar y manipular grandes cantidades de información de manera eficiente.

Organización del Proyecto en GitHub

Los alumnos deben crear un repositorio en GitHub para organizar sus ejercicios y proyectos según las estructuras de control que se han trabajado en clase. El nombre sugerido para el repositorio es: ejercicios_de_clase

Estructura del repositorio:
  • condicionales/ - Todos los ejercicios relacionados con estructuras condicionales.
  • secuenciales/ - Ejercicios que utilizan estructuras secuenciales.
  • bucles/ - Ejercicios relacionados con las estructuras iterativas o bucles.

Por ejemplo, cada carpeta debería contener los archivos .py correspondientes a los problemas o ejercicios de esa categoría.
