Tareas:

  • Crea en GitHub un repositorio llamado: mi_primera_web que contenga un archivo index.html utilizando los siguientes elementos:
    1. Seguir el estándar básico para la estructura de un archivo HTML: 
      
                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Mi Primera Web</title>
                                <meta charset="UTF-8">
                            </head>
                            <body>
                                
                            </body>
                        </html>
    2. Etiquetas obligatorias:
      1. Texto: <h1>, <p>, <br>
      2. Formato de texto: <b>, <mark>, <i>
      3. Listas: <ol>, <ul>, <li>
      4. Tablas: <table>, <tr>, <td>
      5. Medios: <img>, <video>
      6. Formularios: <form>, <input>
      7. Separación de contenido: <div>, <span>
      8. Metadatos: Etiquetas para mejorar la información de la página:
        1. <title> - Título que aparece en la pestaña del navegador.
        2. <meta charset="UTF-8"> - Codificación de caracteres.
        3. <meta name="description" content="Descripción de la página"> - Mejora del SEO con una descripción breve.
        4. <meta name="keywords" content="HTML, web, ejemplo"> - Palabras clave relacionadas con la página.
        5. <meta name="author" content="Tu Nombre"> - Autor del contenido.
        6. <meta name="viewport" content="width=device-width, initial-scale=1.0"> - Responsividad para dispositivos móviles.
    3. Atributos esenciales:
      1. Fuente de medios: src
      2. Hipervínculos: href
      3. Identificación: id (#)
      4. Clases: class (.)
      5. Accesibilidad: alt para imágenes y title para descripciones.
